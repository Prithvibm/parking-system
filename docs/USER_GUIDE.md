# User Guide

## Overview
This guide explains how an end user can use the Parking System application.

## Login
- Open the application.
- Enter your username and password.
- Click **Login** to access the dashboard.

## Dashboard
After logging in, the dashboard provides an overview of the parking system, including available parking slots and recent activity.

## View Parking Availability
1. Navigate to the **Parking Slots** section.
2. View the list of available and occupied parking spaces.
3. Refresh the page to see the latest availability.

## Book a Parking Slot
1. Select an available parking slot.
2. Confirm the booking.
3. A confirmation message will be displayed.

## Logout
Click the **Logout** button to securely end your session.
